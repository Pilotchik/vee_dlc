LMS Виртуальная обучающая среда
=======

Виртуальная обучающая среда для студентов образовательных учреждений.
Включает в себя модули:
  - тестирование;
  - прохождение электронных курсов;
  - анкетирование.

Пример системы на сайте: http://exam.segrys.ru/
